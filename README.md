# DIKWP-NECESSITY²

**无定义结构必然性、偶然触发与十二个月前沿AI精密预测系统**

This project does not claim that contingent history is deterministic. It makes four things deterministic and auditable: the frozen evidence snapshot, the DIKWP×DIKWP relation graph, the probability compiler, and the settlement rule.

## Core correction

- Do not begin by defining “AI progress,” “AGI,” or “intelligence.”
- Ingest observable traces, actor intent fields, constraints, triggers, actions and consequences.
- Treat D/I/K/W/P as co-equal mixed semantic resources.
- Use all 25 ordered transformations.
- Separate structural corridors from surface names and exact dates.
- Preserve unknown axes and change points.
- Pre-register every forecast and settle it publicly.

## Run

```bash
python -m pip install -r requirements.txt
PYTHONPATH=src python -m dikwp_necessity2.cli --trials 20000 --seed 7182026
PYTHONPATH=src pytest -q
```

Open `outputs/dashboard.html`.

## Boundaries

The probabilities are a frozen, inspectable forecast snapshot dated 2026-07-18, not certainty about physical future events and not financial advice. Initial probabilities have not yet accumulated a multi-year settlement record.
